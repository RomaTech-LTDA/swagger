@romatech/swagger

GitHub npm

Zero-config Swagger/OpenAPI documentation generator for Node.js. Uses TypeScript AST analysis to automatically detect routes and generate an OpenAPI 3.0 spec — no decorators, no annotations, no changes to your code. Works with Express, Fastify, and Koa.

Installation

npm install @romatech/swagger

Quick Start

import express from 'express';
import { useSwagger, useSwaggerUi } from '@romatech/swagger';

const app = express();

app.get('/users', (req, res) => {
  res.json([{ id: 1, name: 'Alice' }]);
});

app.post('/users', (req, res) => {
  res.status(201).json({ id: 2, name: 'Bob' });
});

// Two lines — that's it
useSwagger(app);    // → GET /api-docs.json
useSwaggerUi(app);  // → GET /api-docs

app.listen(3000);

Features

Zero Configuration

No decorators, no annotations. Just call useSwagger(app) and your docs are live.

TypeScript AST Analysis

Resolves types from handler signatures and maps them to OpenAPI schemas automatically.

Multi-Framework

Express, Fastify, and Koa. Auto-detects framework from imports and usage patterns.

Multi-Status Detection

Detects all res.status(N).json() chains — generates responses for every status code.

JSDoc Enrichment

@summary, @tag, @deprecated — standard JSDoc enhances your spec.

Swagger UI Built-in

useSwaggerUi(app) serves an interactive documentation page out of the box.
